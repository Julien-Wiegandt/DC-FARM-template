## Fastapi

Get started steps

```
pip install fastapi
```

```
pip install "uvicorn[standard]"
```

```
Create fastapi/main.py
```

```
cd fastapi/
```

```
uvicorn main:app --reload
```

## TypeScript React app

Get started steps

```
npx create-react-app my-app --template typescript
```

```
cd my-app/
```

```
npm install
```

```
npm start
```
